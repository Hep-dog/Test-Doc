Instance.OnlineMonitor.DetectorEventDataMonitor module¶

class Instance.OnlineMonitor.DetectorEventDataMonitor.DetectorEventDataMonitor(appName)[源代码]¶

基类：Core.templateMsgProcess.MsgProcess

Inherited from the template data process class “MsgProcess” This class is used to decode the CSNS detector EventData, and display them in the online website, including:

  1. Pulse ID’s continuity.
  2. Tof Range (0-40)ms, the arbitary scale factor corresponding with detector should be taken as ana
    input parameter “tofUnit”.
  3. Pixel Range (32,32) for monitor and scintillator, (1-60) for short 3He tube, and (1-100) for long one.
